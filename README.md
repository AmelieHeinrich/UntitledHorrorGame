# Untitled Horror Game

This is an attempt at creating a 3D psychological horror game from scratch using no engine and the Odin language.

## ROADMAP

- Audio
- Asset packaging
- World system
- Map editor
- Advanced rendering:
    - Anti-aliasing
    - Color correction
    - Animation
    - Physically based lighting
    - Decals
- Physics
- Make the actual game
